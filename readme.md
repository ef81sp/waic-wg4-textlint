# waic-wg4-textlint

WAIC (ウェブアクセシビリティ基盤委員会) の翻訳ワーキンググループで定められたガイドライン・用語をなんとかする textlint の設定の集合体です。

## 使い方

```sh
# lint only
npx github:ef81sp/waic-wg4-textlint <dir>

# fix
npx npx github:ef81sp/waic-wg4-textlint <dir> --fix
```
